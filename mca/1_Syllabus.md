# BIT Mesra MCA 2026 — Official Syllabus

> **Exam Format:** Online CBT | 100 MCQ | 100 Marks | 120 Minutes | No Negative Marking

---

## Exam Pattern

| Part      | Subject                     | Questions | Marks   | Qualifying Marks   |
| --------- | --------------------------- | --------- | ------- | ------------------ |
| A         | Computer Fundamentals       | 50        | 50      | 35/50              |
| B         | Maths + Reasoning + English | 50        | 50      | No separate cutoff |
| **Total** |                             | **100**   | **100** |                    |

### Selection Process

| Stage        | Marks | Qualifying Marks | Weightage |
| ------------ | ----- | ---------------- | --------- |
| Written Test | 100   | Part A: 35/50    | 40%       |
| Interview    | 100   | 40/100           | 60%       |

- **Merit List Eligibility:** Combined score (Test + Interview out of 100) ≥ 38 AND Rank in top 600

---

## PART A — Computer Fundamentals (50 Marks)

### Section 1: Digital Logic

- Boolean algebra
- Combinational and sequential circuits
- Number representations and computer arithmetic (fixed and floating point)

### Section 2: Computer Organization and Architecture

- Machine instructions and addressing modes
- ALU, control unit
- Instruction pipelining, pipeline hazards
- Memory hierarchy: cache, main memory and secondary storage

### Section 3: Programming and Data Structures

- Programming in C
- Recursion
- Arrays, stacks, queues, linked lists, trees, binary search trees

### Section 4: Algorithms

- Searching, sorting
- Asymptotic worst-case time and space complexity
- Algorithm design techniques: greedy, divide-and-conquer

### Section 5: Operating System

- Processes, inter-process communication, concurrency and synchronization
- Deadlock
- CPU and I/O scheduling
- Memory management and virtual memory

### Section 6: Databases

- ER model
- Relational model, SQL
- Integrity constraints, normal forms
- Transactions and concurrency control

### Section 7: Computer Networks

- OSI and TCP/IP protocol stacks
- Circuit and packet switching
- Framing, IP addressing, IPv4
- Flow control, error control

### Section 8: Software Engineering

- Software Development Life Cycle Models: Waterfall, Prototype, Spiral, Evolutionary, Agile
- Design concepts

---

## PART B — Maths, Reasoning & English (50 Marks)

### Section 1: Mathematics

- Sets, relations, functions
- Combinatorics: counting
- Probability and Statistics: uniform, normal, exponential, Poisson, binomial distributions
- Mean, median, mode, standard deviation
- Conditional probability and Bayes theorem

### Section 2: Analytical Ability & Logical Reasoning

- Logical situations and questions based on given facts

### Section 3: General English

- Comprehension
- Vocabulary
- Basic English grammar (verbs, prepositions, articles)
- Word power, synonyms and antonyms
- Meaning of words and phrases
- Technical writing

---

## Important Notes

- Part A has a **hard qualifying cutoff of 35/50** — must be cleared regardless of total score
- **No negative marking** — attempt every question, never leave blank
- Interview carries **60% weightage** in final merit list — much more than the test
- Exam duration: **120 minutes** = ~1.2 min per question
