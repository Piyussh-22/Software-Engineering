DSA Repository

This repository contains solutions to Data Structures and Algorithms (DSA) problems that I solve.  
The purpose is to strengthen problem-solving skills, build a strong foundation in algorithms, and prepare for coding interviews.

Structure:
- NeetCode/ : Problems based on the NeetCode roadmap
- Leetcode/ : Random LeetCode practice problems

Languages:
Solutions are primarily implemented in Java.

Purpose:
- Improve consistency in solving DSA problems
- Maintain a structured record of progress
- Serve as a reference for future revision and interview preparation

This repository will be continuously updated as more problems are solved.
